Overview

Yeh tools aapke Nrega Bot Desktop Application ke liye zaroori CSV files ko tezi se generate karne mein madad karte hain.

Demand Automation Tools

Applicant List

NREGA portal se save kiye gaye 'Application Register' HTML file ko process karta hai. Yeh tool uss file se sabhi Applicant Names aur Job Card Numbers ko nikaal kar ek saaf CSV file banata hai.

Use: Yeh CSV file aapke Desktop App mein Demand Tab ke liye primary input hai.

Go to Tool

Contractor List

"Applicant List" se banayi gayi CSV file ko upload karke use filter karne mein madad karta hai. Aap poori list mein se apne labourers ko (naam ya job card se) search karke select kar sakte hain.

Use: Yeh tool contractors ko apni team ki ek chhoti, manageable CSV list banane mein madad karta hai, jise Demand Tab mein istemaal kiya ja sakta hai.

Go to Tool

Allocation List

NREGA report se copy kiye gaye raw text block se Work Codes (e.g., 34.../RC/...) ko extract karta hai.

Use: Isse banayi gayi CSV ko Desktop App ke Demand Tab mein 'Load' button se load kiya ja sakta hai taaki 'Auto-Allocate' feature ka istemaal kiya ja sake.

Go to Tool

WC Gen Automation Tools

Scheme List

Yeh ek web-based form (Excel jaisa) hai jisse aap aasani se Work Code generation ke liye zaroori complex "Scheme List" CSV file bana sakte hain. Yeh tool data (khata, plot, village, beneficiary details, etc.) ko sahi format mein taiyaar karne mein help karta hai.

Use: Yahan se generate ki gayi CSV file Desktop App ke WC Gen Tab mein batch automation ke liye input ka kaam karti hai.
